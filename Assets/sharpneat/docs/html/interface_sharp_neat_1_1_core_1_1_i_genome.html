Detailed Description

Generic interface for genome classes.

Concrete IGenome classes are expected to be given a reference to their concrete IGenomeFactory class even if they are spawned from another IGenome. This allows all genomes to use the same set of parameters for spawning - which may change during evolution, e.g. in the case of phased searching in NEAT.
