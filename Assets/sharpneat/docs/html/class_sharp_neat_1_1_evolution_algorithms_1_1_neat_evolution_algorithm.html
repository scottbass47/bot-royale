Detailed Description

Implementation of the NEAT evolution algorithm. Incorporates:

  • Speciation with fitness sharing.
  • Creating offspring via both sexual and asexual reproduction.
Template Parameters
TGenomeThe genome type that the algorithm will operate on.

◆ UpdateBestGenome()

void SharpNeat.EvolutionAlgorithms.NeatEvolutionAlgorithm< TGenome >.UpdateBestGenome ( )
protected

Updates _currentBestGenome and _bestSpecieIdx, these are the fittest genome and index of the specie containing the fittest genome respectively.

This method assumes that all specie genomes are sorted fittest first and can therefore save much work by not having to scan all genomes. Note. We may have several genomes with equal best fitness, we just select one of them in that case.
