Detailed Description

Each instance of ConnectionMutationInfo describes a type of mutation and associated parameters.

A list of ConnectionMutationInfo objects describe a connection weight mutation scheme - a set of a number of different types of mutation along with the probability of each type occurring when spawning offspring genomes asexually.
