\doxysubsection{Detailed Description}
Generic interface for genome classes. 

Concrete \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_genome}{I\+Genome}} classes are expected to be given a reference to their concrete \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_genome_factory}{I\+Genome\+Factory}} class even if they are spawned from another \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_genome}{I\+Genome}}. This allows all genomes to use the same set of parameters for spawning -\/ which may change during evolution, e.\+g. in the case of phased searching in N\+E\+AT.
