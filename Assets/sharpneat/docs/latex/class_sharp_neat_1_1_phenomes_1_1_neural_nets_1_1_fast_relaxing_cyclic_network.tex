\doxysubsection{Detailed Description}
A version of \mbox{\hyperlink{class_sharp_neat_1_1_phenomes_1_1_neural_nets_1_1_fast_cyclic_network}{Fast\+Cyclic\+Network}} that activates a network until it becomes \textquotesingle{}relaxed\textquotesingle{} rather than for some fixed number of iterations. This class is exactly the same as \mbox{\hyperlink{class_sharp_neat_1_1_phenomes_1_1_neural_nets_1_1_fast_cyclic_network}{Fast\+Cyclic\+Network}} in all other respects; See that class for more detailed info. 

A network is defined as being relaxed when the change in output signal value between two successive update iterations is less than some threshold value (defined by max\+Allowed\+Signal\+Delta on the constructor) for all hidden and output neurons (input and bias neurons have a fixed output value).
