Calculates a centroid by comparing each coordinate with every other coordinate. The coord with the lowest average distance from all other coords is the most central coord (the centroid). This method uses an inefficient N$\ast$N comparison of coords to find a centroid. It is provided only as a last resort for distance metrics for which no means exist to calculate a centroid more directly.
