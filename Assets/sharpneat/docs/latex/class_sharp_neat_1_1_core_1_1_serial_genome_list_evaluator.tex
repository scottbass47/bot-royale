\doxysubsection{Detailed Description}
A concrete implementation of \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_genome_list_evaluator}{I\+Genome\+List\+Evaluator}} that evaluates genomes independently of each other and in series on a single thread. 

Genome decoding is performed by a provided \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_genome_decoder}{I\+Genome\+Decoder}}. Phenome evaluation is performed by a provided \mbox{\hyperlink{interface_sharp_neat_1_1_core_1_1_i_phenome_evaluator}{I\+Phenome\+Evaluator}}.

This class evaluates on a single thread only, and therefore is a good choice when debugging code.
