Enable use of hardware accelerated neural network activation functions, i.\+e. alternate implementations that use C\+PU S\+I\+M\+D/vector instructions. 

The vectorized code is provided by alternative classes, and these classes tend to be more complex than their \textquotesingle{}baseline\textquotesingle{} non-\/vectorized equivalents. Therefore when debugging a problem it is often useful to disable use of all vectorized code so as to rule out that code as the source of a problem/bug.
