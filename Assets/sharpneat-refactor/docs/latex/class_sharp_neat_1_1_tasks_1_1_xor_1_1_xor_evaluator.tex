\doxysubsection{Detailed Description}
Evaluator for the logical X\+OR task. 

Two inputs supply the two X\+OR input values.

The correct response for the single output is input1 X\+OR input2.

\mbox{\hyperlink{namespace_sharp_neat_1_1_evaluation}{Evaluation}} consists of querying the provided black box for all possible input combinations (2$^\wedge$2 = 4).
