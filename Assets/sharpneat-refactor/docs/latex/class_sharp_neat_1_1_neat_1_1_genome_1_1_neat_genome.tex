Represents a mapping between genome connection indexes (in \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_genome_1_1_neat_genome_aca8661c21301ed876b3e2d11a730b33a}{Neat\+Genome.\+Connection\+Genes}}), to reordered connections based on depth based node index allocations (as utilised in Acyclic\+Directed\+Graph).

The mapping is in the form of an array of indexes into \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_genome_1_1_neat_genome_aca8661c21301ed876b3e2d11a730b33a}{Neat\+Genome.\+Connection\+Genes}}, i.\+e. the position in the index is the \textquotesingle{}new\textquotesingle{} index (the digraph index), and the value stored at that position is the \textquotesingle{}old\textquotesingle{} index (the genome connection index).\mbox{\Hypertarget{class_sharp_neat_1_1_neat_1_1_genome_1_1_neat_genome_ae68f410b8bb7df69507cd05db7acb371}\label{class_sharp_neat_1_1_neat_1_1_genome_1_1_neat_genome_ae68f410b8bb7df69507cd05db7acb371}}

The directed graph that the current genome represents. 

This digraph mirrors the graph described by \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_genome_1_1_connection_genes}{Connection\+Genes}}; this object represents the graph structure only, not the weights, and is therefore re-\/used when spawning genomes with the same structure (i.\+e. a child that is the result of connection weight mutations only). The Directed\+Graph class provides an efficient means of working with graphs and is therefore made available on this class to provide improved performance for\+:
\begin{DoxyItemize}
\item Decoding to a neural net object.
\item Finding new connections on acyclic graphs, i.\+e. detecting if a random new connection would form a cycle.
\end{DoxyItemize}
