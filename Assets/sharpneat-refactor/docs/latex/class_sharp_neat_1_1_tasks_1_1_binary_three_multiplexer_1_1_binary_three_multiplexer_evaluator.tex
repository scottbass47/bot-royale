\doxysubsection{Detailed Description}
Evaluator for the Binary 3-\/Multiplexer task. 

One binary input selects which of two other binary inputs to output. The correct response is the selected input\textquotesingle{}s input signal (0 or 1).

\mbox{\hyperlink{namespace_sharp_neat_1_1_evaluation}{Evaluation}} consists of querying the provided black box for all possible input combinations (2$^\wedge$3 = 8).
