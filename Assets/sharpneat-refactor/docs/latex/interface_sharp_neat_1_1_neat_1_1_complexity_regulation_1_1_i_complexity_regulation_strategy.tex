\doxysubsection{Detailed Description}
Represents a complexity regulation strategy. 

Determine\+Mode() is called once per generation. A strategy determines the \mbox{\hyperlink{namespace_sharp_neat_1_1_neat_1_1_complexity_regulation_aea67cb189e0a06683f2381679f59b63e}{Complexity\+Regulation\+Mode}} that the evolution algorithm search should be in by examining passed in statistics. As such, the simplest valid strategy is to just return \mbox{\hyperlink{namespace_sharp_neat_1_1_neat_1_1_complexity_regulation_aea67cb189e0a06683f2381679f59b63ea0839c8f3dfc4a4256ae6ff9018158f7f}{Complexity\+Regulation\+Mode.\+Complexifying}}, which results in no complexity regulation taking place.

Complexity regulation is also known as \textquotesingle{}Phased Search\textquotesingle{}. For more information see\+: Phased Searching with N\+E\+AT\+: Alternating Between Complexification And Simplification, Colin Green, 2004 (\href{http://sharpneat.sourceforge.net/phasedsearch.html}{\texttt{ http\+://sharpneat.\+sourceforge.\+net/phasedsearch.\+html}})
