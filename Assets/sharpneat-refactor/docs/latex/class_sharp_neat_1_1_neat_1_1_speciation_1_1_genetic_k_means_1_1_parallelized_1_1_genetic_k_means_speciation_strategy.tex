\doxysubsection{Detailed Description}
A speciation strategy that assigns genomes to species using k-\/means clustering on the genes of each genome. 

This is the speciation scheme used in Sharp\+N\+E\+AT 2.\+x.

This is a multi-\/threaded equivalent of \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_speciation_1_1_genetic_k_means_1_1_parallelized_1_1_genetic_k_means_speciation_strategy}{Genetic\+K\+Means\+Speciation\+Strategy}}, i.\+e. when calling the speciation methods \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_speciation_1_1_genetic_k_means_1_1_parallelized_1_1_genetic_k_means_speciation_strategy_a336a6f55c4820fa83ecb5ab1ea9bdf07}{Speciate\+All()}} and \mbox{\hyperlink{class_sharp_neat_1_1_neat_1_1_speciation_1_1_genetic_k_means_1_1_parallelized_1_1_genetic_k_means_speciation_strategy_aacac000141b970e47a1c1e1738149160}{Speciate\+Add()}}, this class will distribute workload to multiple threads to allow utilisation of multiple C\+PU cores if available.

Multi-\/threading is achieved using the .N\+ET framework\textquotesingle{}s Parallel classes, and thus by default will adjust to utilise however many C\+PU cores are available.
