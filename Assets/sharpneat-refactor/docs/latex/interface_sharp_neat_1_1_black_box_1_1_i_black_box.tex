\doxysubsection{Detailed Description}
Represents an abstract \textquotesingle{}black box\textquotesingle{} function, with and input vector, an \mbox{\hyperlink{interface_sharp_neat_1_1_black_box_1_1_i_black_box_aa1e102d1d6c5d3ea5c371e1669e14545}{Activate()}} method that takes the inputs to produce an output vector. I.\+e. \textquotesingle{}black box\textquotesingle{} here could also be described as a multivariate mathematical function. 

Typically a black box will be a neural network, whereby we set the input vector, activate the network, and read its output vector. However, in principle a black box could be any kind of function or information processing system such as a C\# program or a genetic programming tree.
