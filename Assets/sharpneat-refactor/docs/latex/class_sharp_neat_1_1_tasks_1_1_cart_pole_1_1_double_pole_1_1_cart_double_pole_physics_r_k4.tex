\doxysubsection{Detailed Description}
Represents the cart-\/pole physical model (with two poles); providing a model state update method that employs a classic 4th order Runge-\/\+Kutta to project to the state at the next timestep. 

This cart-\/pole physics code is based on code from \href{https://github.com/colgreen/cartpole-physics}{\texttt{ https\+://github.\+com/colgreen/cartpole-\/physics}} which in turn is based on the cart-\/pole equations from this paper\+: \char`\"{}\+Equations of Motion for the Cart and Pole Control Task\char`\"{} \href{https://sharpneat.sourceforge.io/research/cart-pole/cart-pole-equations.html}{\texttt{ https\+://sharpneat.\+sourceforge.\+io/research/cart-\/pole/cart-\/pole-\/equations.\+html}}
