\doxysubsection{Detailed Description}
For comparing genomes based on their fitness. The comparison result is reversed compared to the standard rules for an I\+Comparer, therefore using this comparer to sort genomes will result in genomes sorted in descending fitness order, i.\+e. fittest genomes first.
