enum SharpNeat.EvolutionAlgorithm.Runner.RunState
strong

An enumeration of possible execution states for an IEvolutionAlgorithm.

Enumerator
NotReady 

Not yet initialized.

Ready 

Initialized and ready to start.

Running 

The algorithm is running.

Paused 

The algorithm has been paused, either due to a user request or because a stop condition has been met. The algorithm can be restarted if the stop condition is no longer true.

Terminated 

The algorithm thread has terminated. The algorithm cannot be restarted from this state, a new algorithm object must be created and started afresh.
