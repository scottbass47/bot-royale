Detailed Description

Wraps a background thread for running an IEvolutionAlgorithm, and methods for asynchronous control of the background thread, for starting, stopping, pausing and restarting the evolution algorithm.

◆ RequestPause()

void SharpNeat.EvolutionAlgorithm.Runner.EvolutionAlgorithmRunner.RequestPause ( )

Requests that the runner pauses, but doesn't wait it to do so.

The runner will pause once the background thread completes execution of the current generation. However, this method returns immediately and does not wait for the background thread to stop.

◆ RequestPauseAndWait()

void SharpNeat.EvolutionAlgorithm.Runner.EvolutionAlgorithmRunner.RequestPauseAndWait ( )

Request that the runner pauses, and waits it to do so.

The runner will pause once the background thread completes execution of the current generation. This method will block and wait for the background thread to stop before returning.

◆ RequestTerminateAndWait()

void SharpNeat.EvolutionAlgorithm.Runner.EvolutionAlgorithmRunner.RequestTerminateAndWait ( )

Request that the runner terminates, and waits it to do so.

The runner will stop once the background thread completes execution of the current generation. This method will block and wait for the background thread to stop before returning.

◆ StartContinue()

void SharpNeat.EvolutionAlgorithm.Runner.EvolutionAlgorithmRunner.StartContinue ( )

Starts the algorithm running.

This method can be called when the runner is in either the Ready or Paused states. If the runner is already in the Running state then the method does nothing and returns (and logs a warning). For all other states an exception is thrown.
