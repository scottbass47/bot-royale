Detailed Description

Evaluator for the Binary 6-Multiplexer task.

Two inputs supply a binary number between 0 and 3; this number selects one of the further 4 inputs (six inputs in total). The correct response is the selected input's input signal (0 or 1).

Evaluation consists of querying the provided black box for all possible input combinations (2^6 = 64).
