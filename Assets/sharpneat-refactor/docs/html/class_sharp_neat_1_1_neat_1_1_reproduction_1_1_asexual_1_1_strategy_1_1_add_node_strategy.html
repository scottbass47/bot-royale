Detailed Description

A NEAT genome asexual reproduction strategy based on adding a single node.

Template Parameters
T

Offspring genomes are created by taking a clone of a single parent genome and adding a single node, if possible. A node is added by selecting a connection at random, and splitting it, i.e. replacing A → B with A → C → B, where A and B are the existing nodes, and C is the new node.
