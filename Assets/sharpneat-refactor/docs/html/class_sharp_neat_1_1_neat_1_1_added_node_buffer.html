Detailed Description

Stores a history of previously added nodes, keyed by the connection that was split to create the node (i.e. the source and target node IDs of the connection).

Used when adding new nodes to check if an identical node (connection split) has been added to a genome elsewhere in the population. This allows re-use of the same innovation ID for like nodes.
