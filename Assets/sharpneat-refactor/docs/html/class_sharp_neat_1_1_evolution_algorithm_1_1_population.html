◆ PopulationSize

int SharpNeat.EvolutionAlgorithm.Population< TGenome >.PopulationSize
get

The number of genomes in the population.

This value is set and fixed to be the length of GenomeList at construction time. During certain phases of the evolution algorithm the length of GenomeList will vary and therefore it may not match PopulationSize at any given point in time, thus this property is the definitive source of the population size.
