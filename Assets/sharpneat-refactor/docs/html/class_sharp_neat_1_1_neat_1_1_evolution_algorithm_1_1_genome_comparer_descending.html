Detailed Description

For comparing genomes based on their fitness. The comparison result is reversed compared to the standard rules for an IComparer, therefore using this comparer to sort genomes will result in genomes sorted in descending fitness order, i.e. fittest genomes first.

Compares two genomes, and returns a value indicating whether one is less than, equal to, or greater than the other.

Parameters
xThe first genome to compare.
yThe second genome to compare.
Returns
A signed integer that indicates the relative values of x and y . If genome x's fitness is higher then genome y's fitness then returns a positive integer. If genome y's fitness is higher then genome x's fitness then returns a negative integer. If the two genomes have equal fitness then returns zero.
